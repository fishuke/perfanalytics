# Perfanalytics Api
PerfAnalytics.API is a restful API which saves data, posted from PerfAnalytics.JS and returns time specific
filtered data.

You can visit demo from [here!](https://perfanalytics-api-fisher.herokuapp.com)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# all tests
$ npm run test:all

# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

```
